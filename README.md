## NFT_TASK

## RUN ON YOUR LOCAL MACHINE 
- `git clone` the repository

- `npm install` to install dependencies
 ```
   $ npm install 
 ```

- `npm start` to start app
 ```
   $ npm start
 ```

 - or you can easily start app with `node server.js` 

 ```
   $ node server.js
 ```
